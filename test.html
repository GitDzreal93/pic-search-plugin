Chrome Extension Test Page

Test English Words

This page contains various English words for testing the word search extension.

Try holding down the modifier key and clicking on words like: beautiful, wonderful, amazing, fantastic.

You can also test with longer words: extraordinary, magnificent, spectacular.

Mixed Content

Here we have mixed content with numbers 123 and symbols @#$% that should not trigger the extension.

But words like computer, technology, and innovation should work perfectly.

Common Words

Test with common words: apple, banana, orange, grape, strawberry.

Animals: elephant, giraffe, butterfly, dolphin, penguin.

Colors: purple, yellow, crimson, turquoise, magenta.

New Site Management Feature

The extension now has a simplified site management system with 3 easy options:

  • 不启用 - Disable the extension on all websites
  • 仅在当前网站启用 - Enable only on the current website (e.g., this page)
  • 在主域名启用 - Enable on the main domain and all its subdomains

Instructions

  1. Install the Chrome extension
  2. Click the extension icon to open settings
  3. Choose one of the 3 site management options
  4. Hold down the modifier key (Command on Mac, Ctrl on Windows)
  5. Click on any English word on this page
  6. The extension should open an image search for that word (if enabled for this site)
